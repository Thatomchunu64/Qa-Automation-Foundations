Project: Employee Management System
Goal

A company has different kinds of employees.

Every employee can:

start working
stop working
display their information

However...

Different employees work differently.

So you'll use an abstract class.

1. Abstract Parent Class
Employee
Attributes
name
employeeId
department

department

Abstract Methods

Notice...

No implementation.

startWork()
stopWork()

🎯 What you're practicing

By the end of this project, you'll have used:

✅ Encapsulation → private attributes.
✅ Inheritance → Developer extends Employee.
✅ Abstraction → abstract Employee class with abstract methods.
✅ Constructors → super(...) to initialize the parent.
✅ Method overriding → each child provides its own implementation of startWork() and stopWork().
✅ Unique child behavior → debugCode() and createLogo().
