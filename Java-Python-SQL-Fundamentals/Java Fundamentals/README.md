# Java Fundamentals

## Overview

This folder contains the Java Object-Oriented Programming (OOP) projects I completed during the first month of my QA Automation Bootcamp.

The goal of these projects is to strengthen my Java programming skills before applying them to automation testing with Selenium WebDriver and other QA automation tools.

---

## Concepts Covered

Throughout these projects I worked on:

- Classes & Objects
- Constructors
- Methods
- Encapsulation
- Inheritance
- Defensive programing
- Access modifiers
- Composition
- Object Interaction
- Problem Solving
- Abstraction
- polymorphism
- SOLID principles
- Collections

---

## Projects

| Project | Description | Key Concepts |
|----------|-------------|--------------|
| Car Management | Simulates managing a car's state and behaviour. | Classes, Constructors, Methods |
| Bank Account | Basic banking system with deposits and withdrawals. | Encapsulation, Validation |
| Computer Manager | Represents computer objects and their attributes. | Objects, Methods |
| Library Book | Models books in a library system. | Inheritance, OOP |
| Smartphone | Simulates smartphone functionality. | Constructors, Object Design |
| Online Store | Basic Online store system- Login and cart functionality | Object state, Constructors, Inheritance, Composition|
